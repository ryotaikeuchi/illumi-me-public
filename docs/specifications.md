# 機能仕様書

## 1. 概要
ユーザーの接近をLINEビーコンで検知し、事前設定された照明パターンでLEDテープを制御するシステム。

## 2. 機能要件

### 2.1 ビーコン検知機能
- **デバイス**: M5Atom Lite
- **検知範囲**: 約5m
- **動作**: 常時ビーコン信号を送信
- **検知方法**: LINEアプリによるビーコン検知

### 2.2 LED制御機能
- **デバイス**: M5Stack Core2
- **LED**: SK6812 LEDテープ（30LED）
- **表示パターン**:
  - フローモード（LEDが流れるように点灯）
  - 点滅モード
- **設定項目**:
  - 色の組み合わせ（ユーザー定義）
- **点灯時間**: システム固定値（1分）
- **エラー時の動作**:
  - デフォルトパターン（レインボー）で点灯
  - デバイスディスプレイにエラー状態表示

### 2.3 ユーザーインターフェース
- **プラットフォーム**: LIFF（LINE Frontend Framework）
- **設定機能**:
  - LED点灯パターン選択
  - カラー設定

## 3. 非機能要件

### 3.1 ハードウェア要件
- 電源: AC電源（常時接続）
- ネットワーク: WiFi接続必須
- 設置環境: 屋内使用

### 3.2 性能要件
- ビーコン検知から点灯までのレスポンス: 3秒以内
- 複数ユーザー同時アクセス: 先着順で処理
